# Retention Agent Copilot Project Overview

Retention Agent Copilot is a portfolio demo for a telecom retention workflow. It helps a call-centre agent understand a customer bill faster, identify why the bill changed, and prepare a concise service note.

## Demo Scope

- Static GitHub Pages demo with synthetic account data.
- Compact bill breakdown for wireless and internet / bundle services.
- Discount detail showing original plan price, savings, and expiry information when visible.
- HUP / device financing detail behind an expandable line-level view.
- Agent Assist panel with call stage, suggested response, follow-up question, and note preview.
- Local development workflow for PDF upload parsing and audio / transcript testing.

## Public Data Boundary

The public demo uses fake numbers, fake account identifiers, and synthetic prices. It does not include real customer PDFs, real call recordings, real account numbers, real bill numbers, real bank payment IDs, or real phone numbers.

## What This Shows

- Product thinking for a real call-centre pain point.
- Frontend workflow design for fast agent scanning.
- Parser and validation logic for telecom-style bills.
- Safety guardrails that keep final offer eligibility in the official system.
- A GitHub Pages portfolio deployment that can be shared publicly.

## What Still Requires A Private / Local Environment

- Real PDF upload analysis.
- Real audio transcription.
- Integration with internal systems such as Maestro, CRM, billing APIs, or offer eligibility tools.
- Production privacy, security, access control, and compliance review.
